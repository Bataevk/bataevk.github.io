МАРТ – ИЮЛЬ 2024

Архитектор и ведущий разработчик RAG-системы | TheWall Global

Разработал архитектуру и вел команду по созданию виртуального ассистента «Ask Handy» на базе RAG с классификатором GLINER. Отвечал за дизайн микросервисов и интеграцию с фронтендом.

В рамках проекта была создана комплексная RAG-система для обработки пользовательских запросов в реальном времени. Ключевые достижения:

  • Спроектировал и реализовал микросервисную архитектуру на FastAPI.
  • Интегрировал NER-модель GLINER для точной классификации намерений пользователя.
  • Настроил гибридный поиск по базе знаний, используя ChromaDB.
  • Обеспечил бесшовную интеграцию с фронтенд-приложением через WebSocket для интерактивного диалога.
